## Beschrijving
<!-- In de Beschrijving staat kort beschreven wat voor project het is en wat je hebt gemaakt -->
De vraag: maak een app zodat de mensen die geinteresseerd zijn om te interageren met de bloemen in de tuin.
volg hierbij het bestaande prototype. (veranderingen mogelijk met overleg).

De opdracht die ik op mezelf heb genomen is de nieuwspagina en een nieuwe interactie, namelijk de suggestions page.
hier kan je een suggestie achterlaten, je voert in wat je favoriete plant is en waarom je deze in de bloementuin zou willen zien.

## Gebruik
<!--Bij Gebruik staat hoe je project er uit ziet, hoe het werkt en wat je er mee kan. -->
Op de app kan je: 
* de artikelen lezen op de bloemennieuws pagina.
* een suggestie achterlaten over wat voor plant je in de tuin zou willen zien. (dit werkt door de POST route in de server.js!)
* naar de kaart om de bloemenopdrachten te maken, hierna unlock je de planten in je collectie en kan je meer informatie vinden per bloem.

## Installatie
<!-- Bij Instalatie staat hoe een andere developer aan jouw repo kan werken -->
1. Clone als eerst deze repository
2. Open hem in VSCodium of een code editor
3. Open dan de terminal en type npm install
4. Start vervolgens de website door npm start in te typen
